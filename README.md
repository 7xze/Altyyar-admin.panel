## نظرة عامة

لوحة تحكم **تيار** الإدارية — واجهة أمامية حديثة مبنية على Vue 3 و Tailwind CSS لإدارة منصة تيار الخلفية (Mastodon). توفر جميع الأدوات اللازمة لإدارة المستخدمين، الخوادم، والإعدادات.

## هيكل المشروع

```
altyyar-admin/
├── src/
│   ├── components/     # المكونات
│   ├── views/          # الصفحات
│   ├── layouts/        # التخطيطات
│   ├── router/         # المسارات
│   ├── stores/         # حالة التطبيق
│   ├── composables/    # دوال مساعدة
│   └── assets/         # الملفات الثابتة
├── public/             # الملفات العامة
├── index.html
├── package.json
├── vite.config.ts

```

## طريقة التشغيل

```bash
# clone المستودع
git clone https://github.com/7xze/Altyyar-admin.panel.git
cd Altyyar-admin.panel

# تثبيت الاعتماديات
npm install

# تشغيل للتطوير
npm run dev

# بناء للإنتاج
npm run build
```
